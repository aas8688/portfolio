Campus Artwork Map

Created using HTML, CSS, JavaScript, Mapbox.js, JSON

This is a collaboration project with journalism students. The project is an interactive map website of twelve artworks locate around campus. To create the interactive map, I use Mapbox.jsto create and configure a campus map with layers, markers and interactivity. By clicking the markers, you can read articles and photos (provided by journalism students) about the artworks.
